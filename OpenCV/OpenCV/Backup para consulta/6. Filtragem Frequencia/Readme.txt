Homorfico:

Para este exercício encontrei e reproduzi a fórmula do filtro homorfico, para isso alterei e adicionei sliders para utilizar as 4 variáveis necessarias.

Removi todos os filtros e códigos desnecessários do dft.

Todos os sliders triggam a função "on_trackbar_height" que utiliza os valores para calcular o filtro.

Também tive que aplicar a função logaritmica na entrada real e a exponencial na saída.

Imagens de exemplo podem ser encontradas na pasta "Images".

Após alguns testes, foi possível observar que cada imagem/iluminação é um caso diferente, ou seja, as variáveis precisam ser ajustadas. E mesmo com uma iluminação extremamente baixa, como na "Image1", foi possível encontrar resultados satisfatórios, enxergando contorno dos braços, detalhes na camisetas, etc.
